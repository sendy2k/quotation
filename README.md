# ใบเสนอราคา Pro — PWA

## วิธีติดตั้งเป็น App

### วิธีที่ 1: GitHub Pages (ฟรี แนะนำ)
1. สมัคร GitHub ที่ https://github.com
2. สร้าง Repository ใหม่ (ตั้งชื่อตามชอบ)
3. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น Repository
4. ไปที่ Settings → Pages → Source เลือก "main branch"
5. รอสักครู่ แล้วเปิด URL ที่ได้ในมือถือ
6. กด "ติดตั้งแอป" หรือ Add to Home Screen

### วิธีที่ 2: Netlify (ลากวางไฟล์)
1. ไปที่ https://app.netlify.com/drop
2. ลากโฟลเดอร์ทั้งหมดวางในหน้านั้น
3. จะได้ URL อัตโนมัติ เปิดในมือถือแล้วติดตั้ง

### วิธีที่ 3: Vercel
1. ไปที่ https://vercel.com
2. ลากโฟลเดอร์ หรือเชื่อมกับ GitHub

---

## ไฟล์ในโปรเจกต์
- `index.html` — แอปหลัก
- `manifest.json` — ข้อมูล PWA (ชื่อ ไอคอน สี)
- `sw.js` — Service Worker (ใช้งานออฟไลน์)
- `icons/` — ไอคอนแอป
